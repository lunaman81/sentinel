**Root cause for Options/Interest = $0:** The `buildNavLines()` function in run.js constructs NAV from the ledger. The ledger provides `stockoptionmarketvalue = -23654.61` and `interest = 527.70`, but `buildNavLines()` only uses `cashbalance`, `stockmarketvalue`, `optionmarketvalue`, and `accruedinterest`. It maps these to the Current Long/Short columns. The issue: `optionmarketvalue` maps to Current Long (max 0) and Current Short (min 0), but the NAV Total line correctly shows `netliquidationvalue = $2,289,421.50` — so the **total is correct even though individual Options/Interest rows show $0**.

This is cosmetic only — NAV total is accurate. The dashboard reads the Total row, not individual asset class rows.

## Section 3: Proposed Fixes (Ranked by Priority)

### P1: Stale option prices from market data snapshots
- **Impact:** Option close prices in CSV can be 20-35% off from broker mark
- **Root cause:** `getMarketData()` returns delayed/stale quotes for illiquid options
- **Fix:** In `buildPositionLines()`, use `position.mktPrice` from the positions endpoint instead of (or as fallback for) the snapshot price. The positions endpoint already has the broker's internal mark.
- **Effort:** Small — change price source in run.js `buildPositionLines()`

### P2: Today's closed trades not reflected until next Flex pull
- **Impact:** Realized P&L understated by $2,327 (INTC close), closed trades count off by 1
- **Root cause:** --live mode preserves Trades/Performance sections from last Flex pull
- **Fix:** Either (a) auto-run `--push` daily to refresh Flex data, or (b) have --live inject Gateway trades into the Trades section. Option (a) is simpler and more reliable.
- **Effort:** Small for (a) — cron job or startup check

### P3: NAV Options/Interest rows show $0 in live merge
- **Impact:** Cosmetic only — NAV total is correct, but individual rows are misleading
- **Root cause:** `buildNavLines()` doesn't map optionmarketvalue or accruedinterest from ledger
- **Fix:** Add `optionmarketvalue` and `accruedinterest` to `buildNavLines()`
- **Effort:** Trivial
